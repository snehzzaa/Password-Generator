# Password Generator

A responsive password generator built with HTML5, CSS3, and vanilla JavaScript.

## Features

- Password length slider from 8 to 32
- Uppercase, lowercase, numbers, and symbols options
- Password strength indicator
- Copy-to-clipboard button
- Responsive layout for mobile and desktop

## Folder Structure

```text
Password-Generator/
├── index.html

```

## How to Use

1. Open `index.html` in a browser.
2. Adjust the length and character options.
3. Click **Generate Password**.
4. Use **Copy** to copy the password to your clipboard.
